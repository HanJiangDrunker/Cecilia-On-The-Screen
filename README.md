# 😀Cecilia-On-The-Screen
屏幕边的塞西莉亚开源啦！  
源代码在Desktop.py  
代码使用了使用PyQt库，记得pip!🫠    
  
目前版本是1.3.5-Release，功能还不是太全。  
感谢开源的AI上色，塞西莉亚变彩色啦！  
我的问题，没写自动适配不同分辨率的代码部分，运行后会显示在屏幕左上角，拖到右下角就可以啦。  
右键塞西莉亚可以隐藏，快速打开网站或者......  
隐藏之后可以在任务栏找到图标，右键显示。
